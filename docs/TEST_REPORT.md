## 四、性能基准测试

**测试数据**：10 万行日志（11.9 MB），6 个服务，混合 INFO/DEBUG/WARN/ERROR 级别。

| 测试项 | 耗时 | 说明 |
|--------|------|------|
| 解析 10 万行日志 | 0.33s | 正则提取时间戳、级别、服务名、内容 |
| 批量写入 10 万条到 PostgreSQL | 1.29s | `cursor.executemany()` 批量插入 |
| 分页查询（第 1000 页，50 条/页） | 0.04s | OFFSET/LIMIT |
| 统计聚合（级别/服务/时间趋势） | 0.09s | GROUP BY + date_trunc |
| 日志列表查询 | 0.007s | 单用户维度 |
| 关键词搜索（ILIKE，命中 20325 条） | 0.08s | 模糊匹配 |
| **端到端总耗时** | **1.83s** | 解析 + 写入 + 查询 |

### 优化记录

| 版本 | 写入方式 | 写入耗时 | 总耗时 |
|------|----------|----------|--------|
| 优化前 | 逐条 INSERT（10 万次数据库往返） | 4.66s | 5.22s |
| 优化后 | `cursor.executemany()` 批量提交 | 1.29s | 1.83s |
| **提升** | | **3.6x** | **2.9x** |
